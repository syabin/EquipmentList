# 设备清单制作工具

## 项目概述

这是一个用于工业设备清单管理的Web工具，帮助工程师快速创建、编辑和管理设备清单。

**核心价值**：将传统的Excel设备清单转化为结构化的Web表格，支持智能搜索、批量操作、工时统计等功能，大幅提升工作效率。

---

## 主要功能

### 1. 设备清单编辑
- 表格式编辑，支持20+字段
- 行拖拽排序（Ctrl+拖拽=复制）
- 批量删除、批量填充
- 右键菜单操作

### 2. 型号智能搜索
- 加载成品型号对照表（Excel）
- 输入型号时自动匹配
- 自动填充设备名称、属性、部门等字段

### 3. 数据导入导出
- **导入Excel**：从已有设备清单导入
- **导出Excel**：生成带格式的Excel文件
- **填合同号**：从RICH定单表自动填充

### 4. 工时统计
- 内置计时器
- 记录工作时间
- 自动保存计时状态

### 5. 撤销/重做
- 支持50步操作历史
- 快捷键：`Ctrl+Z` 撤销，`Ctrl+Y` 重做

---

## 新增功能

### 6. 项目存档系统
- 按项目名称/项目号保存配置
- 存档列表：查看、加载、删除历史项目
- **导出/导入存档**：保存为JSON文件，可备份或迁移

## 技术栈

- **前端**：纯HTML/CSS/JavaScript（无框架依赖）
- **Excel处理**：xlsx.js（SheetJS）
- **存储**：localStorage（浏览器本地存储）
- **兼容性**：Chrome、Firefox、Edge现代浏览器

## 使用流程

### 首次使用
1. 打开 `index.html`
2. 点击「📋 对照表」导入成品型号对照表
3. 开始添加设备行

### 日常使用
1. 点击「+ 添加行」添加新设备
2. 在型号列输入，系统自动匹配并填充
3. 补充其他字段信息
4. 完成后点击「💾 保存」或「📤 导出」

### 存档管理
1. 点击「📁 存档」打开存档面板
2. 输入项目名称，点击保存
3. 历史存档可随时加载或删除
4. 使用「📤 导出全部」备份到文件

## 特殊标记

- **电控硬件**：型号以 `-EE` 结尾
- **上位机**：型号以 `-SE` 结尾
- **普通**：无特殊标记
- **出口**：出口项目
- **医药**：医药行业
- **防爆**：防爆要求
- **有毒**：有毒环境
- **精细化工**：精细化工行业

---

## 注意事项

1. **数据存储**：配置数据保存在浏览器localStorage中，清除浏览器数据会丢失
2. **建议存档**：重要配置请及时存档并导出备份
3. **对照表格式**：导入的Excel需要包含型号、设备名称、属性等列
4. **浏览器兼容**：推荐使用Chrome，旧版IE不支持
